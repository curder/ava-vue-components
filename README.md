# Ava Vue Components 

## Installation

```
yarn add -D @curder/ava-vue-components
```

**Or**

```
npm install --save-dev @curder/ava-vue-components
```


## Register Plugins

```
import '@curder/ava-vue-components/dist/main.css'; // 加载css
import Plugin from '@curder/ava-vue-components';
Vue.use(Plugin);
```
